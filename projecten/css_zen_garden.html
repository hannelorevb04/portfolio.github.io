CSS Zen Garden

In the Design 4 course, we were tasked with designing a CSS Garden, with postmodernism as the assigned theme. The principle of the CSS Zen Garden is to keep the HTML structure unchanged while different designers adjust the page's style by only changing the CSS.

How did I approach it?

  • Exploration:

    In the exploration phase, I began by researching postmodernism and its associated visual style. I studied the key characteristics, colors, shapes, and patterns typical of postmodernist design.

  • Execution:

    The execution phase involved developing and refining the design for the CSS Zen Garden. I started with sketches and wireframes, followed by the actual implementation in HTML and CSS.
